Link da api

A api lista marcas e modelos de carros que são vendidos no mercado brasileiro.
Funçao: Primeiro liste as marcas do tipo de veículo que você deseja, através da ação marcas e sem nenhum parâmetro e na sequencia
você poderá obter a listagem de veículos de uma determinada marca, com o código da marca desejada.

Como os dados foram tratados:

Fiz uma requisição para a API e em seguida armazenei em uma váriavel, depois disso, acessei elas pelo seu 'nome' por exemplo marca.codigo e marca.nome ou seja com isso conseguimos 'pesquisar' dentro da API através do codigo ou nome. depois fiz um HTML pelo JS para acessar essas pesquisar e exibir na tela para o user e ir criando as tags HTML.
